# ChefMind

ChefMind ist Ihr intelligenter Ernährungsbegleiter, unterstützt durch KI. Machen Sie ein Foto oder beschreiben Sie, was Sie gegessen haben, und Gemini AI analysiert sofort den Nährstoffgehalt einschließlich Kalorien, Makronährstoffe, Vitamine und Mineralstoffe.

• KI-Lebensmittelanalyse — Machen Sie ein Foto oder geben Sie eine Beschreibung ein, um eine sofortige Nährwertanalyse zu erhalten
• Personalisierter Wochenplan — Die KI erstellt einen 7-tägigen Speiseplan, abgestimmt auf Ihr Profil, Gesundheitszustand, Allergien und Vorlieben
• Nährstoffverfolgung — Überwachen Sie Vitamine, Mineralstoffe und Makros mit täglichen Referenzmengen
• Einkaufsliste — Fasst automatisch die Zutaten aus Ihrem Wochenmenü zusammen
• HealthKit-Integration — Importieren Sie Alter, Geschlecht, Gewicht und Größe aus der Gesundheits-App
• Intelligentes Ernährungstagebuch — Erfassen Sie Mahlzeiten und verfolgen Sie den Verzehr Tag für Tag
• Premium — Schalten Sie unbegrenzte Menüerstellung, Fotoanalyse und Gerichteaustausch frei

Egal, ob Sie Ernährungsziele verfolgen, neue Küchen entdecken oder einfach neugierig sind, was in Ihrem Essen steckt – ChefMind macht die Nährwertanalyse mühelos.
