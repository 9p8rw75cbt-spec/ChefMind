# ChefMind

ChefMind は、AI を搭載したインテリジェントな栄養管理アシスタントです。写真を撮るか、食べたものを説明するだけで、Gemini AI がカロリー、マクロ栄養素、ビタミン、ミネラルなどの栄養成分を即座に分析します。

• AI 食品分析 — 写真を撮るか説明を入力するだけで、瞬時に栄養分析結果を取得
• パーソナライズ献立 — プロフィール、健康状態、アレルギー、好みに基づいて AI が 7 日間の食事プランを自動作成
• 栄養素トラッキング — ビタミン、ミネラル、マクロ栄養素を 1 日の目標値に基づいて管理
• 買い物リスト — 献立から材料を自動的にまとめる
• HealthKit 連携 — 年齢、性別、体重、身長をヘルスケアからインポート
• スマート食事日記 — 食事を記録し、日々の摂取状況を追跡
• プレミアム — 無制限の献立生成、写真分析、料理の交換をアンロック

食事目標の管理、新しい料理の探求、または単に食べ物の成分に興味がある場合でも、ChefMind は栄養分析を簡単に行えます。
