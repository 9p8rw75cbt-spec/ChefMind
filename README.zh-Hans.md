# ChefMind

ChefMind 是一款基于人工智能的智能营养助手。拍照或描述你吃了什么，Gemini AI 立即分析营养成分，包括卡路里、宏量营养素、维生素和矿物质。

• AI 食物分析 — 拍照或输入描述即可获得即时营养分解
• 个性化周菜单 — AI 根据你的个人资料、健康状况、过敏和偏好生成 7 天饮食计划
• 营养追踪 — 基于推荐摄入量的每日目标，监控维生素、矿物质和宏量营养素
• 购物清单 — 自动整合周菜单中的食材
• HealthKit 集成 — 从健康应用导入年龄、性别、体重和身高
• 智能饮食日记 — 记录餐食并逐日追踪摄入情况
• 高级版 — 解锁无限菜单生成、照片分析和菜肴替换功能

无论你在管理饮食目标、探索新菜系，还是单纯好奇食物中的营养成分，ChefMind 都能让营养分析变得轻松简单。
